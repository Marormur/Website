InstanceManager~InstanceManager(config)

Manages multiple instances of a window type

Constructor

new InstanceManager(config)

Create an instance manager for a specific window type
Parameters:
Name Type Description
config Object Manager configuration
Properties
Name Type Attributes Default Description
type string Window type identifier (e.g., 'finder', 'terminal', 'text-editor')
instanceClass Class Class to instantiate (must extend BaseWindowInstance)
maxInstances number <optional>
 0 Maximum allowed instances (0 = unlimited)
createContainer function <optional>
 Custom container creation function. Receives instanceId, returns HTMLElement
Source:
  • instance-manager.js, line 45
Example
new InstanceManager({
    type: 'terminal',
    instanceClass: TerminalInstance,
    maxInstances: 10,
    createContainer: (id) => { ... }
});
